Инструкция по запуску проекта "Театральная афиша"

1. Требования

- Python 3.11 или новее.
- Установленный pip.
- Браузер для открытия сайта.

2. Установка зависимостей

Откройте терминал в папке проекта и выполните:

python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

Если виртуальное окружение уже создано, достаточно активировать его и проверить зависимости:

.venv\Scripts\activate
pip install -r requirements.txt

3. Запуск сервера

В активированном виртуальном окружении выполните:

python -m uvicorn api:app --host 127.0.0.1 --port 8000 --reload

После запуска сайт будет доступен по адресам:

Главная страница:
http://127.0.0.1:8000/

Личный кабинет:
http://127.0.0.1:8000/account

Административная панель:
http://127.0.0.1:8000/admin

4. Тестовые учетные записи

Обычный пользователь:
логин: user
пароль: user123

Администратор:
логин: admin
пароль: admin123

5. Основные возможности сайта

- просмотр театральной афиши;
- поиск и фильтрация спектаклей;
- просмотр подробной информации о спектакле;
- демонстрационная авторизация;
- выбор мест;
- оформление бронирования;
- отдельный личный кабинет пользователя;
- административная панель;
- добавление, редактирование и удаление спектаклей;
- просмотр бронирований в админке;
- поиск и фильтр заказов;
- пагинация заказов по 20 записей;
- экспорт бронирований в файл .xlsx.

6. Где хранятся данные

Данные проекта хранятся в JSON-файлах:

data/performances.json - список спектаклей.
data/bookings.json - список бронирований.

SQLite в текущей демонстрационной версии не используется.

7. Остановка сервера

Чтобы остановить сервер, нажмите Ctrl + C в терминале, где запущен Uvicorn.
